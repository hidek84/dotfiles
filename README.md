# Dotfiles

## Get started
```
cd .dotfiles
bash deploy.sh
bash initialize.sh
```

## Manual updates
* Sytem Preferences
  * Keyboard
    * Shortcuts
      * Input Sources - disable all
      * Keyboard - update `Move focus to next window`
